Wat is de Functie van een CPU?

Een processor, ook wel bekend als een CPU, is een stuk hardware in een computer. Een CPU verwerkt instructies van Software, Hardware en voert bijvoorbeeld berekeningen uit. Een CPU wordt ook wel gezien als het brein van de computer. Het bestuurt bijvoorbeeld je muis, toetsenbord en nog veel meer dingen. De CPU zegt eigenlijk gewoon wat de computer moet doen.

Alle componenten in een PC worden aangestuurd via de CPU. Je GPU is bijvoorbeeld niks zonder je CPU. Zonder een CPU werkt dus eigenlijk niks. Je hebt ook CPU's waarbij er in de CPU een ingebouwde GPU zit. Dat wordt ook wel APU genoemd. Dit heeft zo zijn voordelen, het is namelijk energiezuiniger en compacter. Een externe GPU zal altijd sneller zijn.

Een processor, ook wel bekend als een CPU is een belangrijk stuk hardware in een computer. De CPU verwerkt instructies van software en hardware en voert bijvoorbeeld berekeningen uit. Daarom wordt de CPU ook wel gezien als het brein van de computer. De CPU bepaalt wat de computer moet doen en wanneer.

De CPU bestuurt onder andere de muis, het toetsenbord en andere onderdelen van de computer. Als je bijvoorbeeld een toets indrukt, stuurt de CPU een signaal om te bepalen wat er moet gebeuren. De CPU zegt eigenlijk constant tegen de computer wat hij moet doen.

De CPU werkt in 3 fasen:
1
Informatie opnemen De CPU ontvangt informatie van de software en hardware.
2
Informatie verwerken De CPU verwerkt de ontvangen informatie en bepaalt wat er moet gebeuren.
3
Informatie uitvoeren De CPU voert de benodigde acties uit, zoals het weergeven van iets op het scherm of het uitvoeren van een programma.

Alle componenten in een pc worden aangestuurd via de CPU. De GPU is bijvoorbeeld niets zonder de CPU. De CPU geeft opdrachten aan de GPU, zoals wat er op het scherm moet worden weergegeven. Zonder CPU werkt dus eigenlijk niks in de computer.

Er bestaan ook CPU's met een ingebouwde GPU. Dit wordt een APU genoemd. Een APU heeft als voordeel dat hij compacter, goedkoper en energiezuiniger is. Dit is handig voor laptops en eenvoudige computers. Een externe GPU is meestal veel sneller en beter geschikt voor zware taken zoals gamen of videobewerking.
